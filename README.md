# Arc Raiders Tracker Data

This folder contains the data files for remote updates.

## Setup Instructions

1. **Create a new GitHub repository**:
   - Go to https://github.com/new
   - Name: `arc-raiders-tracker-data`
   - Make it **Public**
   - Don't initialize with README (we'll upload our own files)

2. **Upload these files** to the repository root:
   - `version.json` - Version metadata
   - `items.json` - Item database

3. **Verify the URLs work**:
   - https://raw.githubusercontent.com/aivarsmeijers/arc-raiders-tracker-data/main/version.json
   - https://raw.githubusercontent.com/aivarsmeijers/arc-raiders-tracker-data/main/items.json

## Updating Data

When game data changes:

1. Run the fetch script to get latest data:
   ```bash
   cd Scripts
   swift FetchItemData.swift
   ```

2. Copy the new `items.json` to this folder

3. Update `version.json`:
   - Increment the version number (e.g., "1.0" â†’ "1.1")
   - Update the `updatedAt` timestamp
   - Update `itemCount` if changed

4. Commit and push to GitHub

5. The app will automatically fetch the new data on next launch

## File Formats

### version.json
```json
{
  "version": "1.0",
  "updatedAt": "2026-01-14T12:00:00Z",
  "itemCount": 495
}
```

### items.json
Full item database - generated by `FetchItemData.swift`
